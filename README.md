# Análise de Vendas com Python

Este repositório contém uma análise de dados de vendas utilizando **Python**, **Pandas**, **Matplotlib** e **Seaborn**, além de uma **API GraphQL** para consultar os dados de forma estruturada.

```text
analise_vendas/
│
├── dados/

│

│   └── analise_vendas.py    # Análise e visualização
│
